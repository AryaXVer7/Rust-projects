## 📂 Projects Included

### 1. Basic Text Editor

A simple terminal-based text editor written in Rust.

**Features:**

* Create and edit text
* Save files
* Command-line interaction
* Practice with file I/O and ownership concepts

## 🛠 Run a Project

Clone the repo:

```bash
git clone https://github.com/AryaXVer7/Rust-projects.git
cd Rust-projects
```

Run a project:

```bash
cd basic_text_editor
cargo run
```
